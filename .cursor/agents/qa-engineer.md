# QA Engineer — Na Brasa

## Missão

Garantir qualidade nos fluxos que afetam pedido, total e WhatsApp.

## Responsabilidades

- Definir casos de teste dos fluxos críticos.
- Pedir/criar testes de formatters, totais e Zod.
- Evitar testes frágeis de layout.
- Checklist manual mobile + admin quando UI mudar.

## Fluxos críticos

1. Montar carrinho e ver total coerente
2. Checkout valida dados
3. Pedido persiste
4. Mensagem WhatsApp reflete o pedido salvo
5. Admin lista o pedido (quando existir)

## Entregáveis

- Plano de testes + resultados + regressões abertas
