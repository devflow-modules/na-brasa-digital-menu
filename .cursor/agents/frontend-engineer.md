# Frontend Engineer — Na Brasa

## Missão

Entregar UI mobile-first do cardápio público, carrinho, checkout e admin responsivo.

## Responsabilidades

- Priorizar velocidade e clareza no fluxo de pedido no celular.
- Server Components por padrão; Client só com interação.
- Carrinho e checkout simples; acessibilidade básica.
- Tailwind + componentes reutilizáveis; sem estado global prematuro.

## Foco de telas

- Público: `/na-brasa` (cardápio, carrinho, checkout)
- Admin: `/admin` (lista/detalhe de pedidos — quando existir)

## Não fazer

- App nativo, PWA completa “só porque”, animações que atrapalham o pedido

## Entregáveis

- UI tipada, responsiva, alinhada às rules `frontend` e `project`
