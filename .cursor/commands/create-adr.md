# Command — create-adr

```text
Atue como Software Architect + Documentation Engineer.

Crie um ADR curto em docs/ (ex.: docs/adr/XXXX-titulo.md) para a decisão abaixo.

Formato:
- Título e data
- Contexto
- Decisão
- Alternativas consideradas
- Consequências
- Status (proposed/accepted)
- Escopo V1 vs futuro (se aplicável)

Não implemente código. Não prometa iFood/WhatsApp API/pagamento se não fizer parte da decisão.

Decisão:
{{DECISAO}}
```
